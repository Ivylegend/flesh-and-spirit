# Icha house of games

Its a combination of multiple gospel based board games. All the distinct board games can be found in the components directory.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```
